## Rss Feed to follow Skiff Blog
The RSS feed is forked from the [RSSHub](https://github.com/DIYgod/RSSHub/). 

## Manual Deployment

Run the following in this directory:
```bash
npm install
npm start
```
Visit http://localhost:1200/skiff/blog to view the feed
